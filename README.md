# 🧮 Calculadora de IMC em Python

Projeto desenvolvido em Python para calcular o Índice de Massa Corporal (IMC) de forma simples e rápida através do terminal.

## 📌 Sobre o Projeto

O objetivo deste projeto é praticar conceitos fundamentais de programação em Python, como:

- Entrada de dados com `input()`
- Conversão de tipos com `float()`
- Operações matemáticas
- Estruturas condicionais (`if`, `elif`, `else`)
- Formatação de saída com `f-strings`

O programa solicita o peso e a altura do usuário, calcula o IMC e exibe a classificação correspondente.

## 🚀 Funcionalidades

✅ Recebe peso e altura do usuário

✅ Calcula automaticamente o IMC

✅ Exibe o resultado com duas casas decimais

✅ Classifica o IMC de acordo com os padrões da OMS

## 📊 Classificações

| IMC | Classificação |
|------|-------------|
| Menor que 18,5 | Abaixo do peso |
| 18,5 a 24,9 | Peso normal |
| 25,0 a 29,9 | Sobrepeso |
| 30,0 a 34,9 | Obesidade Grau I |
| 35,0 a 39,9 | Obesidade Grau II |
| 40,0 ou mais | Obesidade Grau III |
